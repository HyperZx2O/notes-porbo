Q1(a) — Remove Consecutive Duplicate Characters from a String
String Manipulation
▼
Part (a)
Read a string and a character c. Collapse every run of two or more consecutive occurrences of c into a single occurrence.

The program scans the string in-place. When a run of c of length ≥ 2 is found, it shifts the rest of the string left to remove the extras, then decrements the effective length accordingly.

#include <stdio.h>
#include <string.h>

int main() {
    char s[100], c;
    int i, j, count = 0;
    scanf("%99[^\n]", s);
    getchar();
    scanf(" %c", &c);

    int len = strlen(s);
    for (i = 0; i < len; ) {
        if (s[i] == c) {
            count = 1;
            j = i + 1;
            while (j < len && s[j] == c) { count++; j++; }
            if (count >= 2) {
                /* shift everything from j backwards to i+1 */
                for (int k = j; k <= len; k++)
                    s[i + (k - j)] = s[k];
                len -= (count - 1);
            } else {
                i++;
            }
        } else {
            i++;
        }
    }
    printf("%s\n", s);
    return 0;
}
How It WorksWhen a run of c of length ≥ 2 is detected starting at index i and ending before index j, the substring from j onward is copied back to i+1, effectively keeping exactly one c and removing the rest. len is reduced by count−1.

Q1(b) — Recursive Functions & Palindrome Check
Recursion / Strings
▼
Definition

Recursive Function

A function that calls itself to solve a problem by breaking it into smaller, identical sub-problems until a base case is reached. Every recursive function needs a base case to prevent infinite recursion.

Palindrome Check (Recursive Approach)

A string is a palindrome if the first and last characters are equal and the inner substring is also a palindrome.

#include <stdio.h>
#include <string.h>

int isPalindrome(char *s, int start, int end) {
    /* Base case: single character or empty string */
    if (start >= end) return 1;

    /* If mismatch found, not a palindrome */
    if (s[start] != s[end]) return 0;

    /* Recurse for the middle part */
    return isPalindrome(s, start + 1, end - 1);
}

int main() {
    char s[100];
    printf("Enter string: ");
    scanf("%99s", s);
    
    if (isPalindrome(s, 0, strlen(s) - 1))
        printf("1\n"); /* Palindrome */
    else
        printf("0\n"); /* Not Palindrome */

    return 0;
}
Recursive logicThe function isPalindrome checks the characters at start and end. If they match, it calls itself with start+1 and end-1. The recursion stops when start >= end (success) or a mismatch occurs (failure).
Palindrome Examples"racecar" → 1  |  "level" → 1  |  "hello" → 0

Q2(c) — Multidimensional Arrays in C
Concept
▼

Declaration & Memory Layout

A multidimensional array is an array of arrays. Declared as int mat[3][4]; (3 rows, 4 columns). Elements are stored in row-major order — all elements of row 0 followed by row 1, etc. Total memory = rows × cols × sizeof(type).

Initialization

int arr[2][3] = { {1,2,3}, {4,5,6} };  // nested braces (preferred)
int arr[2][3] = {1,2,3,4,5,6};         // linear form (also valid)

Passing to Functions

The column size must always be specified in the parameter:

void func(int arr[][3], int rows);   // valid
void func(int (*arr)[3], int rows);  // also valid
// void func(int arr[][], int rows); // INVALID — omitting column size

Use Cases & Limitations

  • Matrix operations, game boards, image pixel grids, tabular data.
  • Limitation: Fixed size for static arrays; no bounds checking (risk of buffer overflow).
  • Limitation: Complex memory management for higher dimensions.

Q3(c) — Why Must Strings End with '\0'?
Concept
▼

In C, strings are plain arrays of char — there is no built-in length field. The null character '\0' (ASCII 0) acts as a sentinel marking the end. It is essential for three reasons:

  1. Standard library functions such as printf("%s"), strlen(), strcpy(), and strcat() all rely on '\0' to know where the string ends. Without it they read past the intended boundary, producing garbage output, crashes, or undefined behaviour.
  2. Correct length computation: strlen("hello") returns 5 because it counts until '\0'. A missing terminator causes it to scan arbitrary memory.
  3. Security: Omitting '\0' can cause buffer overflows and other memory-safety vulnerabilities.

Q2(a) — Storage Class Specifiers in C
Concept
▼
  • auto — Default for local variables. Allocated on function entry, freed on exit. Scope: block-level. Example: int x; (same as auto int x;)
  • extern — Declares a variable defined elsewhere (another file). Does not allocate memory — only references an existing one. Used to share globals across files. Example: extern int globalVar;
  • register — Hints the compiler to store the variable in a CPU register for faster access. Address cannot be taken (& is illegal). Often used for loop counters. Example: register int i;
  • static — For local variables: retains its value between function calls, initialised only once. For globals/functions: limits scope to the current file (internal linkage).
void func() {
    static int count = 0;  /* persists across calls */
    count++;
}

Q2(b) — Structure vs. Union
Concept
▼

Structure: Groups different data types under one name. Each member occupies its own memory location. Total size ≥ sum of member sizes (plus alignment padding).

Union: All members share the same memory block. Size equals the largest member. Only one member holds a valid value at any time — saves memory.

/* Structure — all members coexist */
struct Student {
    char name[50];
    int  age;
    float marks;
};  /* Size ≈ 58+ bytes */

/* Union — members overlap */
union Data {
    int  i;
    float f;
    char str[20];
};  /* Size = 20 bytes (largest member) */

/* Memory layout:
   Struct:  [name...][age][marks]  <-- separate slots
   Union:   [i / f / str]          <-- same 20-byte block */

Q2(b) — What Does f1() Simulate? Identify & Fix Mistakes
String Functions / Bugs
▼

The function simulates strcat() — it appends string p2 to the end of string p1 and returns the original start of p1.

Bugs in the Original Code

  • char *reserve = *p1; → should be char *reserve = p1; (save the pointer, not the dereferenced char)
  • while(p1 != '\0') → should be while(*p1 != '\0') (dereference to check the character)
  • while(p2 != '\0') *++p1 = ++p2; → should be while(*p2) *p1++ = *p2++; (copy char then advance both)
  • Missing null-terminator *p1 = '\0'; after the copy loop.

Corrected Version

char *f1(char *p1, char *p2) {
    char *reserve = p1;        /* save start of destination */
    while (*p1 != '\0') p1++; /* advance to end of string */
    while (*p2 != '\0')
        *p1++ = *p2++;         /* copy p2 onto end of p1 */
    *p1 = '\0';                /* null-terminate */
    return reserve;
}

typedef struct node { int i; struct node *prev, *next; } node;

void insertAtEnd(node **head, node **tail, int val) {
    node *n = malloc(sizeof(node));
    n->i=val; n->next=NULL; n->prev=*tail;
    if (!*head) *head=*tail=n;
    else { (*tail)->next=n; *tail=n; }
}
void insertAtFront(node **head, node **tail, int val) {
    node *n = malloc(sizeof(node));
    n->i=val; n->prev=NULL; n->next=*head;
    if (*head) (*head)->prev=n; else *tail=n;
    *head=n;
}
void insertAtPosition(node **head, node **tail, int pos, int val) {
    node *temp=*head;
    for (int i=0;i<pos&&temp;i++) temp=temp->next;
    if (!temp) return;
    node *n=malloc(sizeof(node));
    n->i=val; n->next=temp->next; n->prev=temp;
    if (temp->next) temp->next->prev=n; else *tail=n;
    temp->next=n;
}

Q1(b) — Magic Square Validator
2D Arrays
▼

A magic square has equal row, column, and diagonal sums.

int isMagic(int mat[10][10], int n) {
    int sum = 0, d1 = 0, d2 = 0;
    for (int j = 0; j < n; j++) sum += mat[0][j];
    for (int i = 0; i < n; i++) {
        int rs = 0, cs = 0;
        for (int j = 0; j < n; j++) {
            rs += mat[i][j];
            cs += mat[j][i];
        }
        if (rs != sum || cs != sum) return 0;
        d1 += mat[i][i];
        d2 += mat[i][n - 1 - i];
    }
    return (d1 == sum && d2 == sum);
}

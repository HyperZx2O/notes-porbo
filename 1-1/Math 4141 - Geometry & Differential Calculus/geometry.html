01

Standard (Central) Equation of a Sphere

A sphere with centre C(α, β, γ) and radius r. For any point P(x, y, z) on the sphere, CP = r.

Distance Formula → Squaring both sides
√[(x−α)² + (y−β)² + (z−γ)²] = r
⟹ (x−α)² + (y−β)² + (z−γ)² = r²

Expanding and rearranging:

Standard Form
x² + y² + z² − 2αx − 2βy − 2γz + (α²+β²+γ²−r²) = 0
02

General Equation of a Sphere

x² + y² + z² + 2ux + 2vy + 2wz + d = 0

By comparing with the standard form:

u
u = −α
v
v = −β
w
w = −γ
d
d = α²+β²+γ²−r²
Centre and Radius (from general form)
Centre  C ≡ (−u, −v, −w)
Radius  ρ = √(u²+v²+w²−d)
📌 Key Condition: Coefficients of x², y², z² must all be equal, and there must be no xy, yz, zx terms.
03

Sphere Through 4 Points — Determinant Form

Substituting 4 points gives 4 equations in u, v, w, d. Eliminating gives:

x²+y²+z²xyz1
x₁²+y₁²+z₁²x₁y₁z₁1
x₂²+y₂²+z₂²x₂y₂z₂1
x₃²+y₃²+z₃²x₃y₃z₃1
x₄²+y₄²+z₄²x₄y₄z₄1
= 0
04

Plane Section of a Sphere

When a plane (P = 0) cuts a sphere (S = 0), the intersection is a circle.

Circle on a sphere
S = 0  (sphere)    AND    P = 0  (plane)

If OA = sphere radius = ρ, and OC = perpendicular distance from centre to plane:

CA = √(OA² − OC²)  =  radius of circle

Sphere through a circle: Any sphere through the circle (S=0, P=0) is:

Family of Spheres through a circle
S + λP = 0    (λ is a parameter)
05

Intersection of Two Spheres

Two spheres S₁ = 0 and S₂ = 0 intersect in a circle. The plane of that circle is:

S₁ − S₂ = 0    (the radical plane)
📌 Note: S₁ − S₂ = 0 is a plane (x², y², z² terms cancel), not a sphere.
06

Equation of the Tangent Plane

For sphere S = x²+y²+z²+2gx+2fy+2hz+d = 0, tangent at (α, β, γ):

Tangent Plane Formula
xα + yβ + zγ + u(x+α) + v(y+β) + w(z+γ) = 0
📌 Analogy: For circle x²+y² = a², tangent at (x₁,y₁) is: x·x₁ + y·y₁ = a²

Key Concepts

What is a Cone?

A surface generated by a line (generator) passing through a fixed point (vertex) and moving along a fixed curve (guiding curve).

Vertex & Guiding Curve

The vertex is the apex. The guiding curve defines the base shape — circle, ellipse, or hyperbola.

Direction Cosines

A line through vertex (α,β,γ) with direction cosines l, m, n has the symmetric equation below.

02

Core Formulas

General Line Through a Point

(x − α) / l  =  (y − β) / m  =  (z − γ) / n

where (α, β, γ) is the vertex and l, m, n are direction cosines of the generator.

Meeting Point at z = 0

Point  =  ( α − lγ/n ,  β − mγ/n ,  0 )

Substitute into the guiding curve equation to eliminate l, m, n.

Angle Between Two Lines

cos θ  =  (l₁l₂ + m₁m₂ + n₁n₂) / ( √(l₁²+m₁²+n₁²) · √(l₂²+m₂²+n₂²) )

Problem 01

Find the equation of the cone whose vertex is (0, 0, 3) and guiding curve is the circle x² + y² = 4, z = 0.

1
Line through vertex (0,0,3):x/l = y/m = (z−3)/n
2
Where line meets z = 0:x = −3l/n,   y = −3m/n
3
Point must lie on circle x² + y² = 4:(−3l/n)² + (−3m/n)² = 4  ⟹  (l/n)² + (m/n)² = 4/9  — (i)
4
l/n = x/(z−3), m/n = y/(z−3). Substitute into (i):x²/(z−3)² + y²/(z−3)² = 4/9
9(x² + y²) = 4(z − 3)²
Problem 02

The section of a cone with vertex P = (α, β, γ) and guiding curve x²/a² + y²/b² = 1, z = 0, cut by x = 0 is a rectangular hyperbola. Find the locus of P.

1
Line through P = (α, β, γ):(x−α)/l = (y−β)/m = (z−γ)/n
2
Line meets z = 0 at:( α − lγ/n ,  β − mγ/n ,  0 )
3
Point lies on ellipse; substituting l/n and m/n:(1/a²)[α − γ(x−α)/(z−γ)]² + (1/b²)[β − γ(y−β)/(z−γ)]² = 1
4
Simplifying:b²(αz − xγ)² + a²(βz − yγ)² = a²b²(z − γ)²
5
Set x = 0. For rectangular hyperbola, coefficients of y² and z² are equal and opposite. After expanding and applying condition:a²γ² + (b²α² + a²β² − a²b²) = 0  ⟹  α²/a² + (β² + γ²)/b² = 1
Locus of P:   x²/a² + (y² + z²)/b² = 1
Problem 03

Find the lines in which the plane 2x + y − z = 0 cuts the cone 4x² − y² + 3z² = 0. Find the angle between them.

1
Line: x/l = y/m = z/n — (i). Plane: 2l + m − n = 0 — (ii). Cone: 4l² − m² + 3n² = 0 — (iii)
2
From (ii): n = 2l + m. Sub into (iii):4l² − m² + 3(2l+m)² = 0 ⟹ 8l² + 6lm + m² = 0 ⟹ (2l+m)(4l+m) = 0
3
Case 1: m = −2l ⟹ n = 0 ⟹ d.r. = (1, −2, 0) i.e. x/−1 = y/4 = z/2
Case 2: m = −4l ⟹ n = −2l ⟹ d.r. = (1, −4, −2) i.e. x/−1 = y/2 = z/0
4
Angle between (−1,4,2) and (−1,2,0):cos θ = [1+8+0] / [√21·√5] = 9/√105 ⟹ θ ≈ 28.56°
Lines: x/−1 = y/4 = z/2  and  x/−1 = y/2 = z/0  |  Angle ≈ 28.56°

Key Steps — Finding a Cone's Equation

  • Write the generator equation (line through vertex with d.c. l, m, n)
  • Find where the generator meets the plane of the guiding curve (usually z = 0)
  • Apply the guiding curve equation to this meeting point
  • Express l/n and m/n in terms of x, y, z from the line equations
  • Substitute back to eliminate l, m, n — this gives the cone's equation
For the locus problem, set x = 0 and use the rectangular hyperbola condition — the coefficients of the two squared terms must be equal and opposite in sign.

General Equation

The most general second-degree equation in \(x, y, z\) is:

\[Ax^2 + By^2 + Cz^2 + Dxy + Eyz + Fxz + Gx + Hy + Iz + J = 0\]

This can be classified into six standard categories by rotation and translation of axes to eliminate cross-terms.

02

The Six Types of Quadric Surfaces

(i) Ellipsoid
\[\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1\]

Closed bounded surface. Special case a=b=c gives a sphere.

(ii) Elliptic Cone
\[\frac{x^2}{a^2}+\frac{y^2}{b^2}=\frac{z^2}{c^2}\]

Equivalent forms:

\[-\frac{x^2}{a^2}+\frac{z^2}{c^2}=\frac{y^2}{b^2},\quad \frac{y^2}{b^2}+\frac{z^2}{c^2}=\frac{x^2}{a^2}\]

Two conical nappes meeting at vertex (origin).

(iii) Elliptic Paraboloid
\[\frac{x^2}{a^2}+\frac{y^2}{b^2}=cz\]

Opens in one direction; cross-sections are ellipses.

(iv) Hyperbolic Paraboloid
\[cz=\frac{y^2}{a^2}-\frac{x^2}{b^2}\]

Saddle surface — "potato chip" shape.

(v) Hyperboloid of One Sheet
\[\frac{x^2}{a^2}+\frac{y^2}{b^2}-\frac{z^2}{c^2}=1\]

Single connected surface; minimum circle at waist.

(vi) Hyperboloid of Two Sheets
\[-\frac{x^2}{a^2}-\frac{y^2}{b^2}+\frac{z^2}{c^2}=1\]

Two separate sheets; no real points where |z|<c.

03

The Conicoid

A general second-degree surface (conicoid):

\[ax^2 + by^2 + cz^2 + 2fyz + 2gzx + 2hxy + 2ux + 2vy + 2wz = d\]

Central conicoid in standard form:

\[ax^2 + by^2 + cz^2 = 1\]
04

Equation of Tangent Plane

Theorem

Tangent plane to \(ax^2+by^2+cz^2=1\) at \((\alpha,\beta,\gamma)\):

\[\boxed{a\alpha x + b\beta y + c\gamma z = 1}\]

Replace \(x^2 \to x\alpha\), \(y^2 \to y\beta\), \(z^2 \to z\gamma\) in the surface equation (the "half-replacement" trick).

05

Condition of Tangency

The plane \(lx+my+nz=p\) touches \(ax^2+by^2+cz^2=1\) iff:

\[\frac{l^2}{a}+\frac{m^2}{b}+\frac{n^2}{c}=p^2\]

Point of Contact

\[\left(\frac{l}{ap},\;\frac{m}{bp},\;\frac{n}{cp}\right)\]
06

Polar Line w.r.t. the Conicoid

Definition

Polar plane of \((\alpha,\beta,\gamma)\) w.r.t. \(ax^2+by^2+cz^2=1\):

\[a\alpha x + b\beta y + c\gamma z = 1\]

The polar line is the intersection of two polar planes. For direction \((l,m,n)\) through \((\alpha,\beta,\gamma)\):

\[a\alpha x + b\beta y + c\gamma z = 1\]
\[alx + bmy + cnz = 0\]
07

Chord Bisected at a Point (T = S₁)

Chord of \(ax^2+by^2+cz^2=1\) bisected at \((\alpha,\beta,\gamma)\):

\[a\alpha x + b\beta y + c\gamma z = a\alpha^2 + b\beta^2 + c\gamma^2\]

This also gives the cutting plane whose conic has centre at \((\alpha,\beta,\gamma)\).

Solved Problems

Find the equations of tangent planes to the conicoid \(7x^2 - 3y^2 - z^2 + 21 = 0\) which pass through the line \(7x - 6y + 9 = 0,\; z = 3\).
Solution
1

Rewrite: \(7x^2 - 3y^2 - z^2 = -21 \implies \frac{x^2}{-3}+\frac{y^2}{7}+\frac{z^2}{21}=1\)

So \(a = -\tfrac{1}{3},\; b = \tfrac{1}{7},\; c = \tfrac{1}{21}\).

2

Any plane through the given line: \((7x - 6y + 9) + \lambda(z - 3) = 0\)

\[7x - 6y + \lambda z + (9-3\lambda) = 0 \quad\cdots(1)\]
3

Tangency condition with \(l=7,\;m=-6,\;n=\lambda,\;p=3(\lambda-3)\):

\[-3(49) + 7(36) + 21\lambda^2 = 9(3-\lambda)^2\]
\[105 + 21\lambda^2 = 81 - 54\lambda + 9\lambda^2 \implies 12\lambda^2+54\lambda+24=0\]
\[(\lambda+4)(2\lambda+1)=0 \implies \lambda=-4 \text{ or } \lambda=-\tfrac{1}{2}\]
4

\(\lambda=-4\): \(7x - 6y - 4z + 21 = 0\)

\(\lambda=-\tfrac{1}{2}\): \(14x - 12y - z + 21 = 0\)

Answer
\[7x - 6y - 4z + 21 = 0 \quad\text{and}\quad 14x - 12y - z + 21 = 0\]
Find the equation of the tangent plane to \(2x^2 - 6y^2 + 3z^2 = 5\) at the point \((1, 0, -1)\).
Solution
1

Standard form: \(\frac{x^2}{5/2}+\frac{y^2}{-5/6}+\frac{z^2}{5/3}=1\). Point \((\alpha,\beta,\gamma)=(1,0,-1)\).

2

Using tangent formula directly on \(2x^2-6y^2+3z^2=5\):

\[2(1)x - 6(0)y + 3(-1)z = 5\]
\[2x - 3z = 5\]
Answer
\[2x - 3z = 5\]
Find the equation of the polar line of \(-2x = 25y - 1 = 2z\) with respect to \(2x^2 - 25y^2 + 2z^2 = 1\).
Solution
1

Rewrite line in symmetric form:

\[\frac{x}{-1/2}=\frac{y - 1/25}{1}=\frac{z}{1/2}\]

Point \((\alpha,\beta,\gamma)=(0,\tfrac{1}{25},0)\), direction \((l,m,n)=(-\tfrac{1}{2},1,\tfrac{1}{2})\).

2

First polar plane (polar of \((0,\tfrac{1}{25},0)\)):

\[2(0)x + (-25)(\tfrac{1}{25})y + 2(0)z = 1\]
\[-y = 1 \implies y + 1 = 0\]
3

Second polar plane (using direction ratios):

\[2(-\tfrac{1}{2})x + (-25)(1)y + 2(\tfrac{1}{2})z = 0\]
\[-x + 2y + z = 0 \implies x - 2y - z = 0\]
Answer — Polar Line
\[y + 1 = 0 \;\text{ and }\; x - 2y - z = 0\]

\[\frac{3\alpha}{2} = \frac{2\beta}{-2} = \frac{-5\gamma}{-5} = k\]

So \(\alpha = \tfrac{2k}{3},\;\beta = -k,\;\gamma = k\).

3

Constant term condition gives \(k^2=3\). The full system yields:

Verification: \(2(-2)-2(3)-5(-1)+5 = 0\;\checkmark\)
